## Tom's Trivia Game (Contentful API)

A trivia game web app that keeps track of each player's score, announces a winner, and allows you to select the number of questions for a new game.

## Technologies Used:

* HTML5
* CSS3
* Bootstrap
* Javascript
* jQuery
* Contentful API
